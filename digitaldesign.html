Diving into Digital

 I joined Rapidminer Inc. for three months as a UX/UI Design Intern. Although I had been implementing several UX principles during my space design experiences, it was not until this internship that I began engaging with UX design theory in-depth. Working alongside the only other designer at the company, I explored alternative designs for the startup modal for the Studio product, designed and prototyped a more informative support page, and created an exhaustive interaction map for the internal alpha project, a browser-based thin client version of Rapidminer Studio.


Rapidminer Studio Startup Modal Redesign

 After downloading the product, we wanted users to be able to jump in, getting acquainted with the product and its value as quickly as possible. We also wanted to increase engagement with the Rapidminer Community, creating a place for users to engage with each other and the company. To faciliate these goals, we explored an alternative startup modal that presented these opportunities while not detracting from the experienced user's goal when opening the software.


Velox Interaction Map

 As development of the thin client approached the alpha stage, we realized that an interaction map would help us realize the features that we still need to develop before alpha as well as which features we need implemeneted before launch, but would not be in the alpha. This interaction map (click it to see full resolution) allowed us to map out how the various pages played with each other and acted as a checklist as we prepared for alpha release. I developed annotation elements which I then used to annotate every interaction in the client using screenshots of the existing prototype.


Support Page Redesign

 The previous Rapidminer Support page appeared blank if you did not have enterprise support, providing no options for what to do or where to go. This confusing dead end was what we aimed to improve through a support page redesign. After listing the various elements, creating an information hierarchy, I prototyped a page layout using Balsamiq, Sketch, and finally HTML and CSS.
